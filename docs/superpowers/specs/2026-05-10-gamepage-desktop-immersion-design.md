# GamePage 桌面跑团沉浸感优化 — 设计文档

**日期:** 2026-05-10
**范围:** 仅 `GamePage`（跑团界面），`ChatPage`（对话）不改动

---

## 1. 整体桌面布局

```
┌─────────────────────────────────────────────────┐
│  顶部状态栏 (HP/SAN/MP 横条)                     │
├────────────────────┬────────────────────────────┤
│                    │  右侧面板（可折叠）           │
│   对话区（主）      │  - 角色卡                   │
│   - 羊皮纸气泡     │  - 当前状态                 │
│   - AI 文字逐字    │  - 骰子日志（最近10次）      │
│   - 选项按钮       │                            │
│                    │                            │
├────────────────────┴────────────────────────────┤
│  底部输入栏 (+ 命令快捷 + 掷骰按钮)               │
└─────────────────────────────────────────────────┘
```

- 桌面底色：深色毛毡/木纹纹理（CSS radial-gradient 模拟）
- 所有面板"纸片"效果：浅色底 + 微旋转 + 投影 + 羊皮纸噪点纹理
- 窄屏时右侧面板收为底部 tab

## 2. 纸片质感

- 底色 `#f4e4c1`（羊皮纸色），CSS 噪点纹理叠加
- 面板 `rotate(-0.3deg)`，`box-shadow` 模拟纸张投影
- 边缘略带不规则感（border-radius 不同值）
- 面板之间用图钉伪元素分隔

## 3. 掷骰动画

Canvas 独立组件，触发时机：
- 跑团检定时：AI 要求检定 → 输入栏旁出现检定按钮
- 用户手动掷骰：输入栏骰子图标

动画流程（约 2.5 秒）：
1. 骰子从上方落入画面中央
2. 弹跳 2-3 次后停住，正面显示结果
3. 数字放大高亮后缩小移入骰子日志

骰子类型：D100（两个十面骰百分比）、D6、D4/D8/D20

## 4. 骰子日志

右侧面板内，按时间线显示最近 10 条：
- 格式：`[检定类型] → 42/60 ✓成功` 或 `→ 81/45 ✗失败`
- 成功绿色，失败红色

## 5. SAN 值骤降效果

触发条件：单次 SAN 变化 ≤ -9

效果（约 1.5 秒，无声）：
- 屏幕边缘红色光晕淡入淡出
- 画面轻微抖动（±4px，频率递减）
- SAN 数字翻牌动画 + 红色闪烁
- 不做声效

## 6. 掷骰音效

- 骰子碰撞音（0.5 秒 mp3），内嵌 `frontend/public/`
- 仅在掷骰动画开始时播放
- 不做 SAN 变化音效

## 7. 右侧面板

- 默认收起到标签页名，点击展开
- 内容：角色卡概要（属性/技能可滚动）、当前 HP/SAN/MP 数字、骰子日志

---

## 实现方式

| 层 | 技术 |
|---|---|
| 桌面底色纹理 | CSS radial-gradient |
| 纸片噪点 | SVG data URI 纹理 |
| 掷骰动画 | Canvas 组件 |
| SAN 抖动 | CSS `@keyframes` |
| 骰子日志 | React state 数组 |
| 音效 | HTML5 Audio API |
